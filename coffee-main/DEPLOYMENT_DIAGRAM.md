## Безопасность

```
┌─────────────────────────────────────────────────────────┐
│                   SECURITY LAYERS                        │
├─────────────────────────────────────────────────────────┤
│  1. HTTPS (SSL/TLS)          ← Render автоматически     │
│  2. CORS Protection          ← Django настройки         │
│  3. CSRF Protection          ← Django middleware        │
│  4. Token Authentication     ← DRF                      │
│  5. Environment Variables    ← Секреты не в коде        │
│  6. Database Encryption      ← PostgreSQL               │
│  7. Security Headers         ← Custom middleware        │
│  8. CSP (Content Security)   ← django-csp               │
└─────────────────────────────────────────────────────────┘
```
